# Alpha Rent — сайт аренды электровелосипедов в Казани

Многостраничный сайт компании «Alpha Rent»: аренда, ремонт и выкуп
электровелосипедов для курьеров. Чистый HTML/CSS/JS, без сборки и
зависимостей — подходит для любого хостинга и для GitHub Pages.

## Структура

```
alpharent/
├── index.html          Главная
├── arenda.html         Каталог аренды (5 моделей, цены)
├── remont.html         Ремонт техники
├── vykup.html          Выкуп и аренда с выкупом
├── kontakty.html       Контакты, карта, отзывы
├── 404.html            Страница «не найдено»
├── css/style.css       Стили (тема чёрный + красный)
├── js/main.js          Меню, формы, аккордеон, анимации
├── assets/
│   ├── logo.jpg        Логотип компании
│   └── ebike.svg       Иллюстрация электровелосипеда
├── robots.txt          Для поисковых систем
├── sitemap.xml         Карта сайта
└── site.webmanifest    Манифест PWA
```

## Локальный просмотр

Откройте `index.html` в браузере. Для корректной работы карты и
шрифтов лучше запустить локальный сервер:

```bash
python -m http.server 8000
# затем откройте http://localhost:8000
```

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub (например, `alpharent`).
2. Загрузите файлы проекта:
   ```bash
   git init
   git add .
   git commit -m "Сайт Alpha Rent"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/alpharent.git
   git push -u origin main
   ```
3. В репозитории: **Settings → Pages → Source → ветка `main`, папка `/root`**.
4. Через пару минут сайт будет доступен по адресу
   `https://ВАШ_ЛОГИН.github.io/alpharent/`.

## Что заменить перед запуском боевой версии

Сайт собран с учётом будущего домена. После покупки домена и хостинга:

- **Домен.** Везде заменить `alpha-rent-kazan.ru` на ваш реальный домен
  (в тегах `canonical`, `og:url`, `og:image`, в `robots.txt` и `sitemap.xml`).
  Файлы с этими ссылками: все `*.html`, `robots.txt`, `sitemap.xml`.
- **Карта Яндекс.** В `kontakty.html` карта уже настроена на адрес
  «ул. Аделя Кутуя, 110Д, корпус 4». При желании можно заменить на
  точную метку организации (Яндекс Карты → «Поделиться» → «Получить
  код карты»). Инструкция — в комментарии рядом.
- **Отзывы Яндекс.** После регистрации компании в Яндекс Бизнес
  заменить статичные карточки отзывов на официальный виджет отзывов
  (комментарии в `index.html` и `kontakty.html`).
- **Онлайн-оплата.** Формы готовы к подключению платёжного сервиса
  (ЮKassa, Точка, Тинькофф). Место интеграции отмечено комментарием
  в `js/main.js` — после отправки заявки клиент перенаправляется на
  страницу оплаты. Для приёма платежей нужен договор с платёжным
  сервисом и серверная часть (например, на хостинге).
- **Приём заявок.** Сейчас формы показывают сообщение об успехе без
  реальной отправки. Подключите отправку на e-mail/в Telegram через
  скрипт на хостинге или сторонний сервис форм.

## SEO — что уже сделано

- Уникальные `title` и `description` на каждой странице под запросы
  «аренда электровелосипедов Казань» и смежные.
- Семантическая разметка, заголовки H1–H4, `alt` у изображений.
- Микроразметка Schema.org `LocalBusiness` (организация, телефон,
  город, рейтинг).
- Open Graph для красивых ссылок в соцсетях и мессенджерах.
- `robots.txt`, `sitemap.xml`, адаптивная вёрстка, быстрый код без
  тяжёлых библиотек.

## SEO — что сделать после запуска для выхода в топ

1. Добавить сайт в **Яндекс Вебмастер** и **Google Search Console**,
   загрузить `sitemap.xml`.
2. Зарегистрировать компанию в **Яндекс Бизнес** (карточка на Картах) —
   это главный источник трафика и отзывов для локального бизнеса.
3. Собирать реальные отзывы клиентов на Яндекс Картах.
4. Наполнять сайт: фото реальных велосипедов, статьи/блог по теме.
5. Указать одинаковые название, адрес и телефон во всех справочниках
   (2ГИС, Яндекс, Google).
